C++ file designed to be run in a Linux Environment

Compile: type make from the directory containing MCMC.cpp
Run: 	 ./PA5	No arguments

The program will print to stdout the results so ideally the output should be redirected to a file
Ex:	./PA5 > outputfile

This program has hard coded values for CP tables as well as number of iterations to run
Any modifications require altering source code and recompiling.
